# Python-MySQL-School-Project

## About

This is a simple Python and MySQL project made for practicing database connectivity and basic billing system concepts. The program lets you create services, add users, and generate a bill based on the services selected by the user. It was made as a school computer science project, so the focus is more on learning than making a commercial application.

## Features

- Add and remove users
- Create new services
- Load sample users and services
- Generate bills with discounts
- Store data in a MySQL database

## Requirements

- Python 3
- MySQL Server
- mysql-connector-python

## How to Run

1. Make sure MySQL is running.
2. Change the username and password in the code if needed.
3. Install the MySQL connector.
4. Run the Python file.
